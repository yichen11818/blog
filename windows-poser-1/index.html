电脑信息

  • 型号: 惠普 Victus by HP Laptop 16-d1XXX
  • 系统: Windows 11 家庭版 Build 22000

问题描述

发现微软商店无法打开,提示其中一个更新服务未正常运行,代码:0x80070424

错误提示

尝试的解决方法

按下 Windows + R 输入 WSReset.exe 运行

结果:无效

  1. 运行系统疑难解答

    结果:无效

  2. 运行 SFC 扫描

    SFC 命令

  3. 运行 DISM 扫描

    DISM 命令

最终解决方案

通过注册表编辑器手动恢复 Windows Update 服务

具体步骤

  1. 打开注册表编辑器

    Windows + R 输入 `regedit` 运行

  2. 导航到路径:

  3. 找到并处理 wuauserv 项

    • 如果存在,先导出备份后删除
    • 如果不存在,直接进行下一步

  4. 创建修复文件

    详细步骤
    1. 在桌面新建文本文档
    2. 复制修复代码到文档
    3. 将文件后缀改为 .reg
    4. 双击运行
