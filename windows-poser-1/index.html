(已解决)微软商店遇到错误其中一个更新服务未正常运行,代码:0x80070424

发表于2023-08-22|更新于2024-04-26|问题解决
|浏览量:

电脑信息

  • 型号 惠普 Victus by HP Laptop 16-d1XXX
  • 系统 Windows 11 家庭版 Build 22000

1.问题描述

发现微软商店无法打开,提示其中一个更新服务未正常运行,代码:0x80070424

+ R输入 cmd 运行

WSReset.exe
  • 重置商店无效
  • 设置中运行疑难解答无效
  • 运行sfc扫描

+ X windows终端 管理员身份运行

sfc /scannow

  • 运行DISM扫描

    DISM.exe /Online /Cleanup-image /Scanhealth # 扫描你全部系统文件并和官方系统文件对比
    DISM.exe /Online /Cleanup-image /Restorehealth # 把那些不同的系统文件还原成系统官方源文件，跟重装差不多

这时参考Windows Update keeps disabling itself automatically in Windows 11/10和How to perform Clean Boot in Windows 11/10
发现是Windows Update服务无法正常启动

2.解决方法

通过注册表编辑器手动恢复 Windows Update 服务

2.1打开注册表编辑器

+ R输入 regedit 运行

2.2找到计算机\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services路径

2.3找到wuauserv项,如果没有则跳到下一步

右键wuauserv项,选择导出以备份

右键wuauserv项,选择删除
