或hexo博客 如本站

一、静态网页

1.1 注册github账户

github.com 打不开下载 下载链接
进行加速 github

打开文件夹

4.hexo 部署至github

4.1 github sshkey 配置

+ R输入 powershell
powershell中输入

xxx@xxx.com 替换至自己的github注册邮箱
之后会生成一个id_rsa.pub 一般路径在C:\Users\你的用户名\.ssh里
如图

如果找不到.ssh文件夹可以

在 Windows 中查看隐藏的文件和文件夹

4.2 github新建仓库

github上新建一个空仓库

复制这个.git结尾的连接 后续有用

4.2修改_config.yml

打开刚刚初始化的文件夹
找到根目录下的_config.yml

4.3 hexo 渲染部署

vscode打开刚刚初始化hexo的文件夹

ctrl+J

输入hexo g -d 即可渲染并上传至github

5.部署至netlify

https://www.netlify.com/

选择刚刚创建的仓库
部署即可

6.绑定域名
