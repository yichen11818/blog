windows系统设置

发表于2023-09-11|更新于2023-10-30|从零使用windows
|浏览量:

本文介绍 Windows 系统的基本设置和使用技巧，持续更新中。

用户名设置

Windows 是一个多用户系统，每个用户都有独立的：

  • 桌面环境
  • 个人主题
  • 软件配置

系统文件夹名称（如 C:\Users\用户名）一旦设置后需要重装系统才能更改！

微软在线激活

重要提示：

  • 使用微软账号激活时，用户名会自动设置为账号前五位
  • 使用本地账号时，请将用户名设置为英文或数字，避免使用中文（防止某些软件不兼容）

计算机名设置

计算机名用于：

  • SSH 连接
  • 蓝牙设备识别
  • 设备投屏
    建议使用英文命名，避免出现乱码。

关于 C 盘和软件安装

  • 软件安装在 C 盘不会影响系统速度
  • 建议使用软件的默认安装位置，方便管理

关于分区

  • 固态硬盘分区不会提升性能
  • 不分区不影响使用
  • 过度分区可能带来管理困扰

新硬盘分区指南

  1. 打开磁盘管理： + R 输入 diskmgmt.msc
  2. 选择以下方式之一：
    • 新建简单卷：右键未分配空间 → 新建简单卷 → 下一步
    • 扩展现有分区：右键已有分区 → 扩展卷 → 下一步

显示桌面图标

  1. + R 输入：
  1. 选择需要显示的图标

文件显示设置

环境变量配置

三种打开方式：

  1. 设置 → 系统 → 关于 → 高级系统设置 → 环境变量
  2. + S 搜索"编辑系统环境变量"
  3. + R 输入 sysdm.cpl

建议优先修改用户变量，避免修改系统变量

开机启动管理

控制软件开机自启可以：

  • 加快开机速度
  • 减少系统资源占用

请谨慎禁用系统组件的自启动，可能影响系统正常运行

电源管理

  • 笔记本长期插电不会损坏电池
  • 建议保持电量在 95-100% 之间
  • 频繁充放电会降低电池寿命

清灰建议

  • 建议每年进行一次清灰和更换硅脂
  • 出现以下情况时建议清灰：
    • 电脑明显变卡
    • 温度明显升高
    • C 面严重发热

Steam 安全

  • 不要接受陌生好友请求！
  • 不要点击不明链接！
  • 不要在

钓鱼网站识别

域名构成：

  1. 主域名（如 yumi1.top）
  2. 域名后缀（如 .com .cn .top）
  3. 子域名（如 www. blog.）
  4. 路径（如 /link/ /archives/）

安全建议：

  • 注意辨别域名的真实性
  • 可疑网站可以通过以下方式验证：
    • 工信部备案查询
    • VirusTotal 网址检测
