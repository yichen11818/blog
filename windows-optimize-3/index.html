(保姆教程)从零使用windows - 3. 系统设置

发表于2023-09-11|更新于2023-09-16|从零使用windows
|阅读量:

写在最前

本贴持续更新,为个人的一些推荐建议,仅供参考

用户名和计算机名 ⭐️⭐️⭐️

  • 用户名
    win系统是一个多用户系统,每个人都可以使用单独的账号登录win系统。
    于是，对于不同的人，其使用电脑的数据，就会是不一样的，比如电脑桌面、主题、安装的软件等等。
    所以，第一次创建用户的时候，当你使用本地账号或微软账号激活windows时,windows会自动将账户名设置为系统文件夹名称,如C:\Users\用户名

    系统文件夹名称一旦设置后就极难更改,需要重装系统

    微软在线激活
    如果电脑是通过微软账号激活的,则会自动将用户名设置为微软账号的前五位,如C:\Users\11451,此处无影响.如果电脑是本地激活自己设置账号时含有中文的话,一部分对中文兼容不好的软件，在用户目录下写入/读取文件（这部分软件没做中文字符集编码处理），可能会报错。⭐️⭐️⭐️请设置本地账号时 用户名设置为英文或者数字
  • 计算机名
    这个主要用于你的电脑与其他设备的互联上，例如ssh,蓝牙设备，投屏设备。这个名字是可以随意更改的，问题不大，但尽量不要是中文，防止□□□乱码。

关于”C盘战士”与”默认安装位置”

固态硬盘,软件等安装在C盘不会 影响系统速度,安装软件时使用软件的默认安装位置就行 ,方便管理

关于分盘

固态硬盘分盘无法提高性能,不分盘不影响任何使用,分盘反而问题多
如果电脑新装硬盘,可能需要通过以下步骤将新盘格式化并分区
+ R 输入diskmgmt.msc回车
将未分配空间右键选择新建简单卷,一路下一步即可


或者右键你已存在的分区,选择拓展卷,一路下一步即可

桌面显示’此电脑’,’回收站’等图标

+ R 输入cmd以下指令回车


文件显示拓展名与显示隐藏文件夹

软件开机自启管理

关闭软件开机自启动,增快开机速度


如果有下载火绒可在安全工具中进行设置

取消部分系统部件的开机自启可能会导致系统出现异常

关于电源

笔记本长期插电对电池无影响,笔记本自带保护系统
相反,经常充放电极大减少笔记本电池寿命,所以建议长期插电使用,电池电量保持在95-100%之间,延长电池寿命

灰尘清理

建议一年进行一次电脑清灰与更换硅脂,具体可视环境而定,如果电脑没有明显的变卡顿掉帧,c面发烫严重,则无需进行清灰

外接屏幕

笔记本一般自带HDMI接口 当笔记本与外接屏幕连接时,如果想要同时使用笔记本屏幕和外接屏幕,可按下win+p键,选择拓展即可
