从零使用windows

发表于2023-08-07|更新于2023-08-13
|阅读量:

个人的一些推荐建议

安全软件

火绒
干净便捷 安静 可拦截广告弹窗

浏览器

  • Chrome
    界面简洁，占用内存小 部分功能需要vpn才能正常使用
  • Edge
    windows自带浏览器，原生支持
  • Firefox
    开源浏览器，可自定义性强 ,快捷

    浏览器必备扩展

    浏览器如何安装扩展插件
    • 以edge为例 可在右上角->扩展->打开Microsoft Edge加载项 可以在线安装下载拓展
    • 如果想要以文件形式安装拓展，可以在扩展页面打开开发者模式，然后点击加载已解压的扩展程序，选择拓展文件夹即可

      chrome的插件在线下载需要VPN

 

  • AdGuard  
    最强广告拦截 自动屏蔽页面上的所有广告 搜索结果过滤
    遇到部分网站显示因为广告拦截器被拦截的情况，可对此网站设置白名单

 

  • iTab新标签页  
    起始页自定义，可添加书签，快捷方式，天气，时钟，壁纸，搜索引擎等

 

  • wetab起始页  
    同itab起始页,可自定义,各有风味

 

  • 篡改猴 特别推荐   
    通过修改网页的js代码来实现一些功能，比如去除网页的一些元素，修改网页的样式，添加一些功能等
