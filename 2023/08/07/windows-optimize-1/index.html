从零使用windows-1.浏览器及其插件

发表于2023-08-07|更新于2023-08-16|软件推荐
|阅读量:

写在最前

本贴持续更新,为个人的一些推荐建议,仅供参考

浏览器

  • Chrome
    界面简洁，占用内存小 部分功能需要vpn才能正常使用
  • Edge
    windows自带浏览器，原生支持
  • Firefox
    开源浏览器，可自定义性强 ,快捷

浏览器必备扩展

浏览器如何安装扩展插件
  • 以edge为例 可在右上角->扩展->打开Microsoft Edge加载项 可以在线安装下载拓展
  • 如果想要以文件形式安装拓展，可以在扩展页面打开开发者模式，然后点击加载已解压的扩展程序，选择拓展文件夹即可

    chrome浏览器的插件在线下载需要VPN

AdGuard 

最强广告拦截 自动屏蔽页面上的所有广告 搜索结果过滤

遇到部分网站显示因为广告拦截器被拦截的情况，可对此网站设置白名单

iTab新标签页 

起始页自定义，可添加书签，快捷方式，天气，时钟，壁纸，搜索引擎等

wetab起始页 

同itab起始页,可自定义,各有风味

篡改猴 ⭐️⭐️⭐️  

(又名Tampermonkey/油猴) 以下称油猴

通过修改网页的js代码来实现一些功能，比如去除网页的一些元素，修改网页的样式，添加一些功能等

可实现:

b站主页优化

网课刷课时

全局护眼模式

百度网盘限速破解

全网 VIP 视频破解去广告

使用方法

  • 1.获取新脚本
  • 2.搜索链接或关键词
    油猴给出了四个可获取脚本的网站 实际上你可以任何地方找到油猴脚本并添加
    这边推荐第一个 Userscript.Zone搜索 和第二个 GreasyFork(油叉)

油猴脚本推荐

以下插件需要以浏览器安装油猴拓展为前提
跳转到油猴锚点

Bilibili Evolved

  • 清爽主页 栏目去除
  • 主页 直播间去广告
  • 视频下载 夜间模式等可自行探索

FIX for “Bing Search returns to the top”

解决 Bing搜索后，点开一个新标签页，过一段时间后回到搜索结果页，搜索结果页自动回到顶部的bug

Open the F**king URL Right Now

解决类似知乎,QQ,CSDN等网站的外链跳转问题 自动跳转

网盘直链下载助手

各大网盘直链获取插件 搭配idm可实现全速下载

护眼模式

浏览器全局护眼

  • 模式一： 降低网页亮度，对网页颜色影响最小。
  • 模式二： 降低网页亮度，并添加暖色调滤镜，眼睛看起来会比方案一舒服点？我自己是主用这个模式。
  • 模式三： 直接对网页进行负片反色（图片、视频除外），注意：该模式兼容性/通用性一般，部分网页的图片、视频可能不尽人意，难以解决。
    部分网站未生效? 试试对此网站强制开启护眼模式

OCS 网课助手

摆脱学校布置的无用网课 释放垃圾时间(bilibili,MOOC,YouTube上有优质课程资源)

1.打开超星，智慧树，等网课平台，等待页面上出现提示面板(一个白色的小框框，一般在屏幕正中间)
2.点开进行自动播放的相关设置

部分平台检测倍速十分严格,请谨慎开启

3.自动答题

  • 前往 言溪题库 注册账号,并复制题库配置
    复制题库配置


    即可自动答题啦!!

OCS官网链接

https://docs.ocsjs.com/

安装完插件后,可在右上角进行详细设置

如biilibili-evolved中对bilibili开启了夜间模式 可关闭插件:护眼模式
