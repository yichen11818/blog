windows键盘及快捷键

发表于2023-09-17|更新于2023-10-30|从零使用windows
|阅读量:

1.键盘

键盘可如下图所示分为三个区域,功能键区,主键盘区,控制键区

主键盘区
  • Tab 缩进四格,切换输入框
  • CapsLock 切换大小写
  • Shift 按住后可输入大写字母和数字上标点符号
  • Ctrl 配合其他键使用
  • Alt 配合其他键使用
  • Enter 确定
功能键区

位于键盘最上方的一排键，从左到右依次为：ESC F1-F12 PrintScreen ScrollLock PauseBreak

  • Esc键：取消键，位于键盘的左上角。Escaps(取消)的缩写，在许多软件中它被定义为退出键，一般用做脱离当前操作或退出当前运行的软件。
  • 从F1~F12是功能键，F2 重命名文件 F5 刷新 F11 全屏等
  • Pintscreen 截屏键,可以将当前屏幕的内容复制到剪贴板。
  • SrollLock（屏籍滚动显示锁定键）：目前该键已很少用到。
  • Pause或Break（暂停键）：按该键，能使得计算机正在执行的命令或应用程序暂时停止工作，直到按键上任意-一个键则继续。另外，按Ctr+Break键能中断命令的执行或程序的运行。
控制键区
  • Insert(插入键)：按下该键，光标后面的字符将被插入到光标前面的字符中，原来的字符将被后面的字符顶掉。
  • Delete(删除键)：删除光标后面字符
  • Home(起始键)：光标移到此行最前
  • End(结束键)：光标移到行最后
  • PageUp(上翻页键)：光标向上翻
  • PageDown(下翻页键)：光标向下翻
  • 方向键
  • NumLock(小键盘锁定键)：按下该键，小键盘上的数字键将被锁定，只能输入数字，不能输入方向键

2.快捷键

CTRL

ctrl+A 全选

ctrl+C 复制选定项。

ctrl+V 粘贴选定项。

ctrl+X 剪切选定项。

ctrl+Z 撤消操作。

ctrl+Y 恢复操作。

ctrl+F 文档网页内搜索

ctrl+Shift+Esc 打开任务管理器

WIN

+ R 打开“运行”对话框。

+ D 显示和隐藏桌面。

+ G 打开 Xbox Game Bar（可快速切换声道输入输出）

+ P 切换显示设置(多屏幕控制)

+ V 打开剪贴板历史记录。

+     切换输入法

+ Tab 打开任务视图。
