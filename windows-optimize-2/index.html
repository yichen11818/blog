2024年最佳Windows软件推荐：从安全到优化全覆盖

发表于2023-08-15|更新于2024-08-09|从零使用windows

本贴持续更新,为个人的一些推荐建议,仅供参考

必备

[安全防护]火绒

  • 干净 无任何具有广告推广性质的弹窗和捆绑等打扰用户行为
  • 轻巧 占用资源少，不影响日常办公、游戏
  • 简单 一键下载，安装后使用默认配置即可获得安全防护
    一个及其良好安静的安全防护软件

火绒的杀毒只能满足正常使用,并不出色,请勿过度依赖火绒的杀毒系统,而是将火绒当做去广告与关闭windows defender的工具,如果感染病毒较为严重可以试试360及卡巴斯基

同时 如果开启了火绒可以将自带的windows defender关闭 如果不会关那就不管

火绒官网链接

https://www.huorong.cn/person5.html?v=1

[解压软件]7zip

  • 开源 免费 无广告 体积小(不到2MB)
  • 支持你能接触到的所有格式
    下载完成后,你可直接右键需要解压的文件->选择7zip->打开压缩包
    其中,添加:将指定文件进行压缩 提取:将指定文件进行解压缩

你可在压缩格式中选择zip 7z bzip2 gzip tar wim xz等
当然,如果这个压缩包是发给老板的还是推荐使用.zip格式,不然老板就得叫你过来跟你探讨 “小张啊,你刚刚发过来的那个什么.7z打不开了”
同时,找到7zip的页面 右上角工具->选项->7-Zip->添加到右键菜单中

添加到右键菜单中

我仅勾选"打开压缩包",“添加到压缩包”,“提取到当前位置”
同时取消勾选层叠右键菜单可使右键文件时不必多点  

7zip官网链接

https://sparanoid.com/lab/7z/

[steam , github必备]Watt Toolkit

又称steam++,一个多功能 Steam 工具箱

  1. 本地加速
    使 Steam 社区、 GitHub 、谷歌验证码,pixiv等国内难以访问的网页正常访问。使用 Titanium-Web-Proxy 开源项目进行本地反代

使用github copliot的用户,请勿对github api进行加速,否则可能导致github copliot无法正常使用,严重则可能导致封号

2.脚本工坊
通过拦截网络请求提供插件服务

3.账号切换,家庭共享管理
一键账号切换,实现两秒切号,家庭共享顺序切换

4.游戏管理
成就一键解锁,自动挂时长,云存档管理,下载完成后自动关机等

5.自动挂卡
众所周知在steam玩游戏会有卡片掉落,但是单纯挂时长并不会掉落卡牌,使用ASF可以自动掉落卡牌, 这里不展开 ,如有疑惑可在文章下方评论区留言

最后可在Watt Toolkit的设置中设置Watt Toolkit开机自启steam开机自启 ,steam开启不检测更新等

如果觉得steam++好用,可以对steam++的开发者进行捐赠任意金额,同时备注好steam++账号也能去除steam++的广告

2024.4.5更新: 3.0版本极其不好用,建议用2.版本
https://gitee.com/rmbgame/SteamTools/releases/tag/2.8.6

Watt Toolkit官网链接

https://steampp.net/

[下载工具]IDM

下载神器,支持断点续传,多线程下载,windows一般为单线程下载,网速可能只有2~3MB/s,但是多线程下载可以将文件分成多个部分同时下载,这样下载速度可达到你的网速上限

可在选项->连接->最大链接数 可将线程数设置为16或更高

搭配油猴的网盘直链下载助手可实现各大网盘的全速下载,如百度阿里夸克等
详情windows-optimize-1/#网盘直链下载助手

在安装IDM本体时请不要更改默认安装地址,同时允许IDM对浏览器进行安装拓展

[禁用windows自动更新]Wu10Man

摆脱windows自动更新的困扰,禁用windows自动更新
可直接关闭windows更新服务

也可修改暂停时间

同时也可关闭与windows更新有关的注册表

此操作可能会导致微软商店无法正常使用,在使用微软商店前将注册表重新打开即可

图吧工具箱

一款很全的纯净开源的硬件检测工具合集,集成各大常见硬件检测,评分工具
可进行

  • 硬件检测,cpu 内存 显卡 硬盘 的跑分测压测试
  • 硬件信息查看
  • 超频
    比如CPU-Z 可以查看CPU的型号,主频,核心数,缓存大小等信息
    ThrottleStop可对cpu进行超频,降频,降压,提高性能,降低温度,延长电池寿命

    该操作可能会导致电脑不稳定,降低寿命,严重情况下可能会暴毙,谨慎操作

    ADIA64可查看电脑的各项硬件信息,包括cpu,内存,显卡,硬盘,主板,传感器等
    GPU-Z可查看显卡的型号,主频,显存大小,显存位宽,显存类型,显卡驱动版本等信息
    ASSSDBenchMark,Diskinfo,DiskMark可对硬盘进行测速,评分,查看硬盘信息

图吧工具箱还包含硬盘坏道检测,u盘真假容量检测,数据恢复,屏幕鼠标键盘检测,烤鸡工具,windows激活,office激活,启动盘制作等,这里不一一阐述了
